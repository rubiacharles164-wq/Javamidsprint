The program is meant to act as a database for a medication system, holding a superclass for person which is used in both patient
and doctor subclasses, along side the medication and percription classes. these are all used in the medicationtrackingsystem class.
Which is used to edit and delete medications, patients, and doctors. Add patients to doctors list. Search for medication, patients, or doctors.
Generate reports with all classes used, check for expired medication and restock medication.
